## 💽 Installation

🔌 **Backend**
```bash
cd app
```
```bash
composer install
```
```bash
symfony server:start
```

💡 **Frontend**.
```bash
cd app
```
```bash
yarn install
```
```bash
yarn encore dev-server
```
